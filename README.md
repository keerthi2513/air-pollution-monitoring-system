# 🌍 Air Pollution Monitoring System

A web-based application developed using Flask to report, analyze, and visualize air pollution complaints.

## 🚀 Features

- Submit air pollution complaints
- Hotspot detection based on complaint frequency
- Pollution type analysis
- Trend analysis over time
- Interactive charts using Chart.js

## 🛠️ Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Chart.js

## 📊 Modules

- Complaint Submission
- Hotspot Analysis
- Pollution Type Analysis
- Trend Analysis
- Data Visualization
